# サニー - フレンドリーなAIチャットボット

明るくフレンドリーなキャラクター設定を持つAIチャットボットです。

## 特徴

- **キャラクター設定**: 明るくポジティブな「サニー」として会話
- **会話履歴の保持**: セッション内での会話の文脈を記憶
- **シンプルなUI**: 使いやすく見やすいモダンなデザイン
- **レスポンシブデザイン**: スマートフォンでも快適に利用可能

## 技術スタック

- **バックエンド**: Flask (Python)
- **LLM**: OpenAI GPT-4o-mini
- **フレームワーク**: LangChain
- **フロントエンド**: HTML, CSS, JavaScript

## セットアップ

1. 必要なパッケージをインストール:
```bash
pip install flask python-dotenv langchain langchain-openai
```

2. `.env`ファイルにOpenAI APIキーを設定:
```
API_KEY=your_openai_api_key_here
```

3. アプリケーションを起動:
```bash
cd 16_llmapp/original
python app.py
```

4. ブラウザで `http://localhost:5000` にアクセス

## 使い方

1. テキストボックスにメッセージを入力
2. 「送信」ボタンをクリック、またはEnterキーを押す
3. サニーが親しみやすく回答します
4. 「会話をクリア」ボタンで会話履歴をリセット可能

## キャラクター設定

サニーは以下の特徴を持っています：
- 親しみやすく、ポジティブな言葉遣い
- 丁寧で分かりやすい回答
- 適度な絵文字の使用
- 専門的な質問にも対応可能
- 分からないことは正直に答える
